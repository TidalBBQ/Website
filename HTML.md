# HTML
HTML超文本標記語言(**H**yper **T**ext **M**arkup **L**anguage)\
透過標籤的方式來呈現網頁\
例如:
```html
<p>我的第一個網頁</p>
```
## 常見且常用的標籤
- h1~h6\
   **h1**為最大的依序遞減下去至**h6**
   
- 表單(form)\
  表單用於把使用者輸入的資料收集起來\
  有許多種輸入方法
  - 文字方塊
  - 單/多選項按鈕
- 表格(table)
  表格可以把資料用更方正的方式呈現\
First Header | Second Header
------------ | -------------
Content from cell 1 | Content from cell 2
Content in the first column | Content in the second column
